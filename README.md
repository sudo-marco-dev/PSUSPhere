# PSUSphere Project

PSUSphere is a Django-based application for managing student organizations, colleges, and programs.

## Features
- **College Management**: Track colleges.
- **Program Management**: Track academic programs linked to colleges.
- **Organization Management**: Manage student organizations.
- **Student Management**: Track students and their programs.
- **Membership**: Manage student memberships in organizations.
- **Admin Interface**: Customized Django admin for easy management.

## Setup
1. Create a virtual environment and activate it.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run migrations: `python manage.py migrate`.
4. Run server: `python manage.py runserver`.
